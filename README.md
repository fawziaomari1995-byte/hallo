# موقع فعاليات محافظة درعا

دليل تثبيت وتشغيل بسيط لمشروع موقع فعاليات محافظة درعا.

## متطلبات

- Node.js مثبت على الجهاز
- npm متاحة من خلال Node.js

## خطوات تشغيل المشروع

1. افتح نافذة الطرفية (Terminal) في مجلد المشروع `hallo`.
2. ثبّت التبعيات:

```bash
npm install
```

3. شغّل الخادم:

```bash
npm start
```

4. افتح المتصفح وادخل إلى:

```text
http://localhost:3002
```

## بيانات الدخول الجاهزة

يمكنك تسجيل الدخول مباشرة باستخدام أحد الحسابين التاليين:

- يوجد حسابات مُعدة مسبقًا في النظام للمدير والمستخدم العادي، لكن يُفضل إنشاء حساب جديد لحماية الخصوصية.

## تجربة مستخدم جديد

يمكن لأي شخص إنشاء حساب جديد من صفحة الموقع عبر زر "إنشاء حساب".

## ملاحظات

- المشروع يستخدم قاعدة بيانات SQLite في المجلد `db/events.db`.
- يوفر النظام الآن حجز مقاعد تفاعلي (VIP، عام، خلفي) على خريطة مقاعد مبرمجة.
- يوفر المدير لوحة تحليلات لرصد مبيعات التذاكر، الإيرادات، ونسب الحضور لحظياً.
- تولد التذاكر رمز استجابة سريعة (QR) فريد لكل حجز، ويمكن تنزيلها كملف PDF.
- يمكن إعداد فعاليات هجينة (Hybrid Events) مع تذاكر افتراضية وحضور عن بُعد.
- إذا أردت إرسال المشروع لشخص آخر، أرسل المجلد كاملًا مع ملف `package.json` وملف `server.js`.
- لا تحتاج إلى إعداد مفتاح API للخرائط، حيث يتم استخدام OpenStreetMap وLeaflet.

## نشر المشروع على استضافة مجانية

يمكنك نشر المشروع بسهولة على خدمة مجانية تدعم Node.js مثل Render.

1. أنشئ حسابًا مجانيًا في `render.com`.
2. ارفع المشروع إلى مستودع GitHub.
3. في Render اختر `New` ثم `Web Service`.
4. اختر المستودع الخاص بالمشروع.
5. ضع إعدادات البناء:
   - Build Command: `npm install`
   - Start Command: `npm start`
6. اضغط `Deploy`.

> لقد تم تعديل `package.json` بحيث يستخدم `node server.js` مباشرة، وهذا يسمح للخدمة بتعيين المنفذ (`PORT`) تلقائياً.

### ملاحظات مهمة

- المشروع يستخدم SQLite محليًا في `db/events.db`.
- على خدمات الاستضافة المجانية قد تكون ملفات البيانات مؤقتة أو تُفقد إذا أُعيد نشر التطبيق.
- إذا أردت أن يعمل البريد الإلكتروني فعليًا في الإنتاج، فعليك إضافة متغيرات بيئة SMTP مثل `SMTP_HOST` و`SMTP_USER` و`SMTP_PASS`.
- يمكنك استخدام `APP_URL` لتعيين عنوان الموقع الكامل إذا احتاج التطبيق لذلك.

إذا تريد، أساعدك خطوة بخطوة في رفع المشروع على Render أو Replit بعد أن ترفع ملفاتك إلى GitHub.
